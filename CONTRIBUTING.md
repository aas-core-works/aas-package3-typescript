# Contributing

We welcome contributions to aas-package3-typescript.

Please see the docs in [doc/contributing](doc/contributing/intro.md) for detailed guidance on workflow, style, testing, CI, and releasing.

## Quick Summary

1. Fork the repository (or create a feature branch if you are a member)
2. Implement your changes
3. Run checks locally:

   ```bash
   npm ci
   npm run typecheck
   npm run lint
   npm test
   ```

4. Commit with clear messages
5. Open a pull request

## Questions

Open an issue if you want to discuss a change before implementing it.
